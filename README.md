# 🛡 Exam Safe Journey

A student-safety-first exam travel support platform designed for competitive exam aspirants (SSC, RRB, UPSC, State Exams).

## 🎯 Mission

Reduce last-minute exam travel panic and improve safety — especially for female aspirants.

This is NOT a social chat app.

This is a structured, safety-focused infrastructure system.

---

## 🧩 Core Features

- Exam Centre Search
- Safety Snapshot (Lighting, Police Presence, Crowd)
- Verified Travel Tips
- Safe Stay Listings
- Female-Only Journey Matching
- Auto-Expiring Communication
- Admin Moderation Dashboard

---

## 🏗 Architecture

### Frontend
- Next.js 14
- TypeScript
- TailwindCSS

### Backend
- NestJS
- PostgreSQL
- Redis

### Admin Dashboard
- Next.js (RBAC enabled)

---

## 🛡 Safety First Principles

- No public phone numbers
- No WhatsApp/Telegram links
- No open group chats
- AI-based moderation
- Audit logs for government review
